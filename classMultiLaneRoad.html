Computes the time evolution of the road and outputs the whole data to a file in CSV format.

The CSV file contains a column for the time (starting at 0) and 3 columns for each Car with lane, velocity and location.

It does not catch any exceptions from std::ofstream, though it does check if it could open the file.

Parameters

stepsAmount of integration steps

Returns
0 Everything good
